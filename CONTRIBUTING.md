First off, thank you for considering contributing to the **NEURAL LINK Ecosystem**. We welcome help to make this P2P network more secure, faster, and more immersive.

### Reporting Bugs
If you find a glitch in the matrix (a bug), please open an **Issue** on GitHub. Include:
1. A description of the bug.
2. Steps to reproduce it.
3. Browser and device information.

### Suggesting Enhancements
Have an idea for a new 3D animation or a security feature?
* Open an **Issue** with the tag `enhancement`.
* Describe your idea and how it fits the "Sci-Fi/Cyberpunk" aesthetic.
